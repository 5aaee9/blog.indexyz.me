Shadowsocks的正确食用方法

时间: 2017-08-09 分类: Usage  

最近开了一个ShadowSocks的服务器，然而发现很多的童鞋都不会用SS科学上网 我只能说:“Naive!”

ShadowSocks简介

Shadowsocks（中文名称：影梭）是使用Python等语言开发的、基于Apache许可证开源的代理
软件。Shadowsocks使用socks5代理，用于保护网络流量。在中国大陆被广泛用于突破防火长城（GFW），以浏览被封锁的内容。 From Wikipedia

那么问题来了，我们要怎么使用SS进行科学上网呢

使用ShadowSocks科学上网

获取软件

首先，这是一个开源项目，托管在GitHub上,因此我们可以很方便的进行获取源代码和协助开发 > Tips: ShadowSocks的作者曾经被请去喝茶，所以我们现在看到的主分支是rm,只要切换到master就好了

Windows的项目: Widnows Android的项目: Android IOS的项目: IOS

单击项目上的Release,可以找到最新的构建版本，我们就可以用它来进行我们的科学上网了,当然,你也可以自己下载master分支的源代码进行构建

Android

可以通过扫描二维码来快速添加服务器

然后返回到主界面 就可以链接服务器了, Android 自身会报警,同意就可以了

然后就可以畅享自由的互联网了

Windows

链接服务器

获取你自己Shadow的账号和密码 如果你的服务器有提供二维码进行输入的话 可以通过状态栏,服务器,扫描二维码来进行扫描

这时候,你的SS已经链接到服务器了, 但是我们还需要使用一个客户端, 链接到SS上，进行科学上网

方法一:使用全局PAC

ShadowSocks自带了从GFW List获取PAC的方法 只需要在状态栏图标右键,PAC,从GFWList更新本地PAC就可以了

这时候默认的IE浏览器就可以访问自由的互联网了

高级篇——自定义PAC

待填坑

方法二:使用浏览器插件

Chrome

安装插件 Proxy SwitchyOmega: 谷歌商店地址 GitHub

然后打开拓展的配置页 添加一个规则, 这里叫做ShadowSocks

修改它为本地的代理服务器

然后点击任务栏图标, 单击刚刚添加的规则就可以科学上网了

FireFox篇

安装附加组件AutoProxy

然后打开附加组件的控制GUI(重启后) 添加代理服务器

端口填入本地端口

现在选择全局代理模式就可以跨越防火长城了

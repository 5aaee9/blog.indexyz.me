Node.js的Base64解码和编码

时间: 2017-08-09 分类: Node  

最近在写一个开车软件小程序, 用上了Base64, 然而发现 Node.js 中并没有相关的内置包. 看了下文档发现其实已经包含在Buffer里了

编码

1const str = new Buffer('Strings');
2console.log(str.toString('base64'));
3// -> U3RyaW5ncw==

解码

1const str = new Buffer('U3RyaW5ncw==', 'base64');
2console.log(str.toString());
3// -> Strings

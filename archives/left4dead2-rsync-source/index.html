Left 4 Dead 2 服务器Rsync同步源

时间: 2017-08-09 分类: Game  

最近因为L4D2的某些原因(Downtown插件需要GLIBC2.15以上然而CentOS6无法提供), 重装了服务器, 然而阿里云上的SteamCMD老是拉不下来端, 于是我在国外的文件服务器上搭建了Rsync源来解决服务端问题

使用方法

1yum install rsync
2// 链接密码root
3rsync -avzP --delete root@rsync.indexs.lol::l4d2server serverfiles

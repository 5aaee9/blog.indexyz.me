利用 Socat 单端口转发加速TCP链接

时间: 2017-08-09 分类: Linux  

有童鞋说我的Minecraft服务器链接速度很慢 我是托管在国内的Azure上的 既然这么说了 那我就拿 Socat 转发到阿里云上

安装Socat

CentOS

1yum install -y socat

Ubuntu

1apt-get update
2apt-get install -y socat

使用

1nohup socat TCP4-LISTEN:本地端口,reuseaddr,fork TCP4:远程地址:远程端口 >> 
2/root/socat.log 2>&1 &
3iptables -I INPUT -p tcp --dport 本地端口 -j ACCEPT
4service iptables save
5service iptables restart

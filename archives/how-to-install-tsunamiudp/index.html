在Linux上安装TsunamiUDP进行快速传输

时间: 2017-08-09 分类: Linux  

TsunamiUDP 是一个专为网络加速诞生的小工具。 思路很简单，使用TCP进行传输控制、UDP进行数据传输。

安装

1# 安装依赖
2yum install autoconf automake gcc unzip
3# 下载源代码编译
4wget https://github.com/cheetahmobile/tsunami-udp/archive/1.8.1.zip
5unzip 1.8.1.zip
6cd tsunami-udp-1.8.1
7./recompile.sh
8make install

开启输出服务器

1cd /data
2tsunamid *

链接服务器获取数据

1cd / data
2tsunami
3> connect {serverip}
4Connected.
5> get *
